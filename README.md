# Updating-and-deploying-the-PMB-2-robotic-platform-to-a-ROS2-version.
This Master’s Thesis addresses robotic obsolescence by migrating the PMB-2 robot from ROS 1 to ROS 2. Validated with SLAM and Nav2 for map generation and navigation, this work gives a robust platform a "second life" and delivers a structured, replicable workflow to modernize similar legacy platforms.
